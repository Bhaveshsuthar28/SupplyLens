4. Run the application

## For Repository Maintainers

### How to Create the Release:

1. **Create a ZIP file** of the datasets:
   - Go to the `Dataset/` folder
   - Select all 4 Excel files
   - Right-click → "Send to" → "Compressed (zipped) folder"
   - Rename to: `SupplyLens-Datasets.zip`

2. **Create GitHub Release:**
   - Go to: `https://github.com/YOUR_USERNAME/SupplyLens/releases`
   - Click "Draft a new release"
   - Tag: `v1.0.0`
   - Title: `Sample Datasets`
   - Description: `Sample supplier order datasets for SupplyLens`
   - Upload the `SupplyLens-Datasets.zip` file
   - Click "Publish release"

3. **Update this file** with your actual GitHub username

Your shareable link will be:
```
https://github.com/YOUR_USERNAME/SupplyLens/releases/download/v1.0.0/SupplyLens-Datasets.zip
```

Share this link anywhere - it's a direct download!
